# BiblioVip - Plataforma de Acesso a Livros

## Tecnologias Utilizadas
- HTML5
- CSS3 (Variaveis CSS, Gradientes, Animacoes)
- JavaScript (Countdown, Notificacoes, Contadores)
- Squeleton CSS Framework
- WOW.js (Animacoes de scroll)

## Funcionalidades Principais
- **Timer de Urgencia**: Countdown regressivo para criar senso de urgencia
- **Catalogo de Livros**: Cards interativos com livros disponiveis
- **Sistema de Notificacoes**: Notificacoes em tempo real de "reservas" de outros usuarios
- **Programa de Fidelidade**: Sistema de bonus VIP com gamificacao (5, 10, 20 livros)
- **Estatisticas Animadas**: Contadores que animam ao entrar na viewport
- **Modal de Reserva**: Pop-up de confirmacao de reserva
- **Depoimentos**: Secao de prova social com avaliacoes

## Design
- Paleta de cores: Azul (#3b5fe8), Ciano (#14a5e9), Verde (#10b981)
- Layout responsivo
- Efeitos de hover e transicoes suaves
- Background com gradientes e ondas SVG
